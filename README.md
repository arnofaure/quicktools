# Quick Tools

A 100% free, single-page collection of small local browser utilities. Nothing you load ever leaves your browser — everything runs client-side.

## Tools

- **Compare Images** — load a before/after image and drag the handle to compare them.
- **Image to GIF** — turn a sequence of images into an animated GIF, with optional MP4 conversion (via ffmpeg.wasm).

More tools will be added over time via the sidebar.
